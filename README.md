# Glassmorphic To-Do List

현대적이고 세련된 Glassmorphism 디자인이 적용된 웹 기반 할 일 목록 애플리케이션입니다. 사용자 친화적인 인터페이스와 브라우저 로컬 저장소를 활용한 데이터 지속성을 제공합니다.

## ✨ 주요 기능
- **할 일 추가:** 새로운 할 일을 입력하고 즉시 목록에 추가할 수 있습니다.
- **완료 표시:** 할 일 항목을 클릭하여 완료 상태를 토글합니다.
- **삭제:** 항목 오른쪽의 `×` 버튼을 눌러 목록에서 삭제할 수 있습니다.
- **데이터 보존:** `localStorage`를 사용하여 브라우저를 새로고침하거나 닫아도 데이터가 유지됩니다.
- **실시간 통계:** 남은 할 일의 개수를 하단에서 실시간으로 확인할 수 있습니다.

## 🚀 실행 방법
이 프로젝트는 정적 웹 애플리케이션으로 별도의 빌드 과정이 필요 없습니다.
1. 저장소를 클론하거나 ZIP 파일을 다운로드합니다.
2. `index.html` 파일을 웹 브라우저(Chrome, Edge, Safari 등)에서 엽니다.

## 🛠 기술 스택
- **HTML5:** 시맨틱 태그 기반의 구조 설계
- **CSS3:** Glassmorphism 효과(`backdrop-filter`), 변수(`:root`), 애니메이션 적용
- **JavaScript (Vanilla JS):** 외부 라이브러리 없이 순수 JS로 비즈니스 로직 구현

## 📂 프로젝트 구조
- `index.html`: 메인 UI 구조 및 요소 정의
- `style.css`: Glassmorphic 디자인 시스템 및 레이아웃 스타일링
- `script.js`: 할 일 관리 로직 및 로컬 스토리지 연동
- `GEMINI.md`: 프로젝트 개발 가이드 및 상세 컨텍스트

## 📝 개발 규칙
- **디자인:** `backdrop-filter`와 `rgba` 색상을 적극적으로 활용하여 유리 질감 효과를 유지합니다.
- **데이터 구조:** 할 일 데이터는 `{ id, text, completed }` 객체 배열 형태로 관리됩니다.
- **코드 스타일:** 가독성을 위해 기능별로 함수를 분리하고, DOMContentLoaded 이후에 초기화 로직을 실행합니다.
